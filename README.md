# 🤖 AI Chatbot using Flask + Ollama

A simple **local AI chatbot web application** built using Flask (Python) and Ollama.  
It connects a web UI (HTML/JavaScript) with a local AI model running on your system.

---

## 🚀 Features

- 💬 Chat with AI from browser
- ⚡ Fast local responses (no OpenAI API needed)
- 🧠 Powered by Ollama models (llama3 / tinyllama / gemma)
- 🌐 Simple web UI (HTML + JS)
- 🔗 Flask backend API

---

## 🛠️ Tech Stack

- Python 🐍
- Flask 🌐
- Ollama 🧠
- HTML + JavaScript 💻
